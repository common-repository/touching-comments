走心评论，精选评论。每一条评论，都是一个故事！

== Description ==

大家在不同的时间，不同的故事中，发出一条评论。

或许就是这么一些不经意的话语给当下你一丝温暖和感触。

管理员可选择的将用户在文章下的评论标记为走心评论，并可以在独立页面上展示这些入选的走心评论。

这便于我们以后访问时重拾当时的心境与温暖。

== Installation ==

= 从WordPress中安装 =

1. 访问 **插件 > 添加插件**。
2. 搜索 **走心评论**。
3. 安装并启用插件。

= 手动安装 =

1. 下载 **走心评论**。
2. 将整个 `touching-comments` 文件夹上传到 `/wp-content/plugins/` 目录。
3. 从您的 **插件** 页面启用 **走心评论**。

== Frequently Asked Questions ==

= 独立页面的走心评论分页点击无效怎么办？ =

分页功能的使用依赖 Wordpress 启用评论分页功能，如果插件分页点击不生效，检查 **仪表盘 - 设置 - 讨论 - 其他评论设置 - 分页显示评论** 前面的复选框是否已勾选。

== Screenshots ==

1. 通过 设置 -> 走心评论设置 来管理。
2. 点击爱心图标将这条评论加入走心评论。
3. 这条评论已入选走心评论。
4. 编辑器添加简码的方式。
5. 在独立页面展示入选的走心评论。
